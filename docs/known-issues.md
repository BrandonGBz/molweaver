# Known Issues

- PyMOL pip installation may fail on Windows because of native DLL issues.
- Conda or micromamba installation is recommended.
- Headless Linux or macOS runners may need Xvfb or OpenGL support for full ray-traced rendering.
- Linux support needs more testing.
- Local file paths are OS-specific.
- CI does not install PyMOL yet, so real render tests are manual/local.
